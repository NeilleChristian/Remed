Domain
VS URL

URL terdiri dari protokol, domain, dan path atau jalur. Protokol menunjukkan bahwa website memiliki sertifikat SSL. URL hanya akan memiliki path kalau mengarahkan pengunjung ke halaman tertentu sebuah website. Jadi Domain adalah nama situs, sementara URL akan mengarahkan pengguna ke salah satu halaman dalam situs tersebut.

Pengertian
Hosting

Hosting web adalah layanan yang menyimpan situs web atau aplikasi web Anda dan membuatnya mudah diakses di berbagai perangkat seperti desktop, seluler, dan tablet.

Fungsi
Hosting

1. Penyimpanan Data Situs Web

Hosting menyimpan semua file yang dibutuhkan untuk menjalankan situs web

2. Akses Internet

Hosting memberikan server yang dapat diakses oleh pengguna melalui internet, dan memungkinkan situs web atau aplikasi dapat dilihat oleh orang lain di seluruh dunia.

3. Keamanan

Hosting juga menyediakan berbagai tingkat keamanan untuk melindungi situs web dari ancaman, seperti serangan malware atau peretasan.
